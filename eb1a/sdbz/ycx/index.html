美国EB-1A移民十大标准之原创性的重要贡献

您一定见过很多推荐美国EB-1A杰出人才项目或者EB-2 NIW国家利益豁免项目的，都会这么说：“只要您满足杰出人才十条中的三条要求，或者NIW中高学历以外六条中的三条就一定符合项目要求！”

果真如此吗？

事实上，比满足这些标准更为重要的是，您呈现给移民官的故事是否有内核。即：在您专业领域内，什么技能、成就让您引以为傲，您是怎么做到的，您个人的独特价值是什么，又能为美国带来怎样的遐想？提炼不出故事的灵魂，只是干巴巴的凑条件，成功率必然不高。

高度概括的官方要求，我们要从哪些方面来论证？

我们可以看到在这两个高级别人才项目中移民局对于“独特价值”强调了三个方面，“原创性“”对行业的“贡献”以及“重要影响力”。围绕这三个论证方面，我们需要在如下方面展开论述:

  1. 您的贡献必须是原创性的，它可以是一种创新的方法、一种有突破性的理念、一种更新的技术等，不是简单复制别人的工作经验，要能和同行区分开来；
  2. 原创的贡献不能只停留在实验或者刚刚结题的阶段，还必须有实际的应用；
  3. 原创性贡献的意义不能只停留在为一个单位、公司层面，还必须有行业重要意义，给行业领域带来改变；
  4. 原创性贡献还需要有影响力，比如得到了同业或者异业的关注、引发了公众的评论、媒体的关注等。

通过常见申请人背景，分析论证“独特价值”的思路

1.高校教授、科研人员

对于这类申请人，常见的原创性成就证据有：科研学术论文、出版物、专利、受基金资助的课题等。

  • 科研论文

在EB-1A的申请中，论文和出版物首先会被归类为“出版物”资质来申报，但是如果申请人的某些论文，能体现他多年原创性研究成果的内核或突破性成就，并且文章发表在国内外影响因子较高的重要刊物上，发表后有较多的同行研究人员引用和讨论，甚至激起了媒体和公众的关注和讨论，这种论文或出版物，就可能被视为申请人对所在领域做出了“重大贡献”。

我们可以用在线引文搜索查询申请人文章的引用情况，大多数高校图书馆科技查新中心也可以提供这样的服务。常见的引文检索工具包括：google scholar，Scopus，SciFinder，Science Index A和ISI科学网，从这类引文检索网站打印出引文索引是非常有价值的证据。

  • 专利

专利的原创点不难证明，难的是如何证明它的行业重要价值；有不少高校的专利还停留在实验室阶段，但专利技术在产业中的实际应用及创造的经济利益和社会利益才是审核人员最关注的，因此我们需要重点提供：

  • 专利应用证明
  • 专利转让合同
  • 专利产品销售的协议、合同
  • 专利销售范围、销售额、产量的说明等

  • 一些证明专利行业先进性及认可度的文件，如：专利相关的媒体报道、奖项、收到的政府补贴、拨款或者其他奖励。

  • 科研课题：

承担科研课题是高校老师和科研人员工作的一个重要组成部分，然而实际案例中，不是所有申请人的科研课题都被审核人认可为“原创性重大贡献”。我们发现，科研课题在以下情况下被认可为原创贡献的概率更高：

  • 课题的研究内容符合美国的经济、文化、教育、福利利益
  • 研究项目得到了国家或地方权威机构的资助，申请人是项目的负责人和主要贡献者
  • 研究的同行评议意见反馈信或科技成果鉴定报告中，明确提到申请人的研究“有原创性”、“创新性突出”、“国际先进”、“国内领先”、“前沿”、“关键技术”、“学术价值高”“建议优先资助”等字眼
  • 研究成果获得了一些行业公认的奖项，或发表在该领域重要刊物上
  • 研究成果得到了实际的应用，在行业内树立了一些新的技术标准或者标杆。

2.企业技术高工、高管

对于这类申请人，我们常见的原创性成就证据有：设计作品，专利，承担的企业科研课题，发表的学术论文、出版物，行业标准制定等。专利、论文、科研课题的证明和上述相同，我们重点说说设计及行业标准制定。

  • 设计作品

对于建筑师、电气工程师、钢铁设计师、照明设计师等设计类出身的申请人，设计作品、图纸、规划方案等都可以视为原创性证据。难点依然是如何证明其有“重大”价值。我们通常从下面入手：

  • 设计作品、图纸中能体现申请人为作品的主要设计人、团队关键贡献者
  • 设计作品被项目发布方实际采纳并应用，项目已完工
  • 设计作品获得了国际、国家或行业权威机构颁发的奖项
  • 设计作品获得了媒体关注和报道，如地标建筑、规模建筑群、重要公共场所，项目甲方是国际国内知名机构

  • 设计人在设计中和国际国内知名机构、知名设计师共同合作

  • 行业标准制定

我们接触到的很多企业高工类客户，或多或少都有参加一些国家及行业标准的制定工作的经历，这些标准明确提到了申请人是主要贡献人或起草人。我们也可以申报成“原创贡献”，并提供：

  • 标准的发布方为国家部门或全国性行业组织的证据
  • 标准发布方出具的证明，证明标准制定的意义，参与标准制定的专家的资质要求，邀请申请人参与的原因
  • 同期其他标准起草人介绍，证明标准的权威性及申请人的行业内权威地位
  • 标准发布后的社会影响，对行业进步及标准化的贡献

3.医学研究者、医学院教授

对于这类申请人，常见的原创性成就证据有：承担的科研课题，发表的学术论文、出版物，发明专利等。证据的提供，和上述科研人员大体相同，但是我们需要在这类客户的申请中突出下面几点：

  • 讲明申请人研究工作的具体领域的重要价值，比如所主攻领域或疾病的患者人数，消耗的社会医疗成本，目前疗法普遍存在的问题或未攻克的难点
  • 突出申请人的研究成果如何使得较大的患者群体受益，如申请人发现了什么新的病理机制，攻克了那些行业难题，开创了什么新的疗法，或者改进了那些治疗方案，参与研发了那些创新药品和医疗器械，取得了那些显著的临床成果，对患者带来了那些好处。

4.艺术家、艺术培训老师

对于艺术家，常见的原创性成就证据有：艺术作品（画作、影视剧作、歌曲作品等）及创造的新学派、曲派或画派等；

对于艺术培训老师，其实教授的学生就是他们最大的作品，我们通常会将他们独特的教育培训观念、方法凝练成原创点，将教授的学生成就作为原创贡献的应用成果及价值体现。通常举证的着力点包括：

  • 强调发表作品的艺术平台或媒体平台，主办方的权威程度
  • 艺术作品的国际国内获奖情况，指导学生的作品获奖情况
  • 媒体或专业出版物对艺术家或者学生作品的报道及评价
  • 教授的学生或学生家长出的证明信

5.运动员、教练

对于运动员，虽然每个都有自己独特的运动技巧，但是我们往往比较难证明他的竞技成就归功于他独特的运动技巧，因为运动员的成就除了天赋，更多来自团队的力量。

但对体育教练类的申请人来说，原创性贡献就是非常重要的一项申请资质，我们往往从下面几个方面着力：

  • 创新独特的教练方法、技巧、团队管理能力的总结和凝练
  • 同行对这种独特训练方法的评论及赞誉
  • 媒体对教练成就的报道
  • 教练本人因教练工作获得的重要奖项
  • 所带运动员或学生的体育竞技成就、奖项
  • 运动员或学生的证明

6.企业家、职业经理人

对于这类商业背景客户，鲜少见到技术专利或创新、论文成就，更多的独特价值体现在：比如建立了一套创新性的商业模式、地产开发模式，发起国内有影响力的商业平台，从而取得了商业上的成功。

但这点在举证上相对比较复杂，需要个案分析及媒体报道、出版物、奖项证据的共同作用。

通过上述从官方要求、到论证方法解析，并综合我们在不同领域申请人的实操干货，大家其实可以感受到一点：无论你是什么背景的的申请人，你所呈报的专业会员、专著或论文、同行评审经历等等，都是需要围绕自己的独特价值而展开的。

在我们的工作程序中，会和每位想要评估是否满足美国EB-1A或者NIW项目的客户，深度沟通其工作的点滴、创业的国王、成就的细节，目的就是希望，把看似无关的一些条件和论据综合起来，挖掘每个申请人的独特价值，并展开有关 “原创性重大贡献”的证明。
